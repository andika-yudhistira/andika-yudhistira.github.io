About Me

Halo semuanya, perkenalkan nama saya Andika Yudhistira, saya adalah seorang IT Guy, Content Creator, dan juga Virtual Drifter. Saya seorang lulusan Teknik Informatika dari kampus hijau Bina Insani yang berada di Bekasi. Saya pernah memiliki pengalaman bekerja di bidang IT Support selama 2 tahun 2 bulan sebagai Helpdesk di PT. Gunung Raja Paksi. Jobdesk yang saya lakukan yaitu melakukan troubleshooting komputer, menangani problem langsung dengan user, dan memantau kinerja server.

Kegiatan saya saat ini adalah membuat konten hobi yang saya upload ke Youtube. Disaat ada waktu luang saya juga menyempatkan diri untuk belajar dan menambah ilmu untuk memperdalam kemampuan programming saya. Bahasa program yang saat ini saya kuasai adalah VB.net, Javascript dan PHP. Sementara untuk skill networking saya juga menguasai Cisco dan Mikrotik. Saya juga mampu untuk melakukan troubleshooting komputer dan melakukan editing video.
